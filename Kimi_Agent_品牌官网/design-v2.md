# Landon 品牌官网设计文档 V2

## 设计概述
- **风格**: 苹果风格毛玻璃科技感
- **目标市场**: 东南亚 (新加坡、马来西亚、泰国、印尼、越南)
- **语言**: 中英文双语
- **主色调**: 品牌深蓝 #0B2676

## 品牌色彩
- **主色 (Foundation Blue)**: #0B2676
- **辅助色 (Sky Blue)**: #B1E8FE
- **深色背景 (Deep Ocean)**: #040B35
- **浅色背景**: #F5F7FA
- **文字主色**: #1A1A2E
- **文字副色**: #6B7280

## 设计特点

### 1. 毛玻璃效果 (Glassmorphism)
- 导航栏: `backdrop-filter: blur(20px)` + `background: rgba(255,255,255,0.7)`
- 卡片: `backdrop-filter: blur(16px)` + 半透明背景
- 按钮: 悬浮时毛玻璃效果

### 2. 动画效果
- 页面滚动视差
- 元素淡入淡出
- 悬浮微动效
- 粒子背景

### 3. 布局
- 全屏Hero区域
- 三栏服务展示
- 数据统计区域
- 联系表单
- 页脚
